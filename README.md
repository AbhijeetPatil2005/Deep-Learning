# Deep Learning Course

Welcome to the Deep Learning course repository. This repository contains code, notes, and resources for various deep learning topics, structured progressively from the fundamentals to advanced neural network architectures.

## Repository Structure

The course is organized into the following modules:

- **01_Perceptron**: Introduction to the fundamental building block of neural networks.
- **02_MLP (Multi-Layer Perceptron)**: Exploring feedforward artificial neural networks.
- **03_Backpropagation**: The core algorithm for training neural networks.
- **04_ANN (Artificial Neural Networks)**: General applications and implementations of ANNs.
- **05_CNN (Convolutional Neural Networks)**: Architectures designed for image and spatial data processing.
- **06_RNN (Recurrent Neural Networks)**: Architectures for sequential data such as time series and natural language.
- **07_LSTM (Long Short-Term Memory)**: Advanced recurrent networks designed to mitigate the vanishing gradient problem.

*(Note: This course is currently a work in progress. More topics and learnings will be added as I continue the course!)*

## Setup

The repository uses Python environments for managing dependencies (`venv` or `tf_env` if you are specifically using TensorFlow).

To activate the virtual environment (Windows):

```powershell
.\venv\Scripts\Activate
# or
.\tf_env\Scripts\Activate
```

*(Make sure you have the required packages installed, typically via a `requirements.txt` if available, or manually install libraries like TensorFlow, PyTorch, Keras, etc. as needed per module).*

## Usage

Navigate to the specific module directory to explore the code examples and Jupyter notebooks provided for each topic.
